## Setup
To run this project, clone the repo:

```
$ export SPOTIPY_CLIENT_ID="Your spotify client id"
$ export SPOTIPY_CLIENT_SECRET="your client secret"

$ export FLASK_APP=app.py FLASK_DEBUG=1
$ python3 -m flask run
```
